# Фронтенд

Стек: React + Redux Toolkit + Material-UI

## Инструкция по настройке и запуску для разработчика

Это руководство описывает процесс настройки рабочего окружения для frontend-приложения с нуля на операционной системе Windows.

### Шаг 1: Установка необходимого ПО

#### 1.1. Git

Установите Git для Windows. Инструкции и установщик доступны на официальном сайте:

* [git-scm.com](https://git-scm.com/download/win)

#### 1.2. Node.js

Установите Node.js напрямую с официального сайта.

1. Перейдите на сайт [nodejs.org](https://nodejs.org/).
2. Скачайте установщик для **LTS** (Long-Term Support) версии. Это рекомендуемая версия для большинства проектов из-за ее стабильности.
3. Запустите скачанный `.msi` файл и следуйте инструкциям установщика, оставляя все опции по умолчанию.
4. После установки откройте **новый** терминал (PowerShell или Командную строку) и проверьте, что все установилось корректно:

    ```shell
    node -v
    npm -v
    ```

    Вы должны увидеть номера установленных версий.

#### 1.3. pnpm

Установите менеджер пакетов `pnpm` с помощью `npm`.

```shell
npm install -g pnpm
```

Для проверки введите `pnpm -v`.

### Шаг 2: Клонирование и локальная настройка Git

1. Откройте терминал и клонируйте репозиторий из закрытой организации:

    ```shell
    git clone https://github.com/UnNamedCodersRvIT/smart-warehouse-rtk
    ```

2. Перейдите в папку с проектом:

    ```shell
    cd smart-warehouse-rtk
    ```

3. **Переключитесь на ветку `develop`.** В нашем проекте вся новая разработка ведется в ветке `develop`, или в ветках от неё, а `main` используется для стабильных версий. Перед началом работы всегда переключайтесь на `develop`.

    ```shell
    git checkout develop
    ```

    Чтобы убедиться, что вы находитесь в нужной ветке, выполните `git branch`. Активная ветка будет отмечена звездочкой (`*`).

4. **Настройте имя и email для коммитов (локально).** Этот шаг гарантирует, что все ваши коммиты в этом проекте будут правильно привязаны к вашему GitHub-аккаунту.

    * **Найдите ваш `noreply` email на GitHub:**
        1. Зайдите на [github.com](https://github.com) и откройте `Settings` (Настройки).
        2. В меню слева перейдите в раздел `Emails`.
        3. Убедитесь, что стоит галочка `Keep my email addresses private`.
        4. Скопируйте email, который выглядит примерно так: `12345678+username@users.noreply.github.com`.

    * **Выполните в терминале (в папке проекта) следующие команды:**

    ```shell
    # Установить ваше имя
    git config --local user.name "Ваше Имя или юзернейм"

    # Установить ваш noreply email
    git config --local user.email "ID+username@users.noreply.github.com"
    ```

    *Флаг `--local` означает, что эти настройки будут применяться **только** для текущего репозитория.*

### Шаг 3: Установка зависимостей проекта

Теперь перейдите в папку с frontend-частью и установите зависимости.

```shell
cd frontend
pnpm install
```

Эта команда скачает все необходимые для проекта библиотеки в папку `node_modules`.

### Шаг 4: Запуск проекта в режиме разработки

Для запуска локального сервера выполните команду:

```shell
pnpm run dev
```

В терминале появится адрес для локального доступа, обычно `http://localhost:5173/`. Откройте этот адрес в браузере.
